# express-react-app
This is getting started with express and react

# Heroku Deployment checklist -
1. Dynamic port binding - 

    `const PORT = process.env.PORT`
2. Specify node engine in package.json file

    `"engines": {
        "node": "8.1.4",
        "npm" : "5.0.3"
    }`
3. Start script -

    `node index.js`
